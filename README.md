# MCrates
<p>MCrates is a cloning plugin from PiggyCrates, Make simple and customizable crates plugin, supporting an unlimited amount of crate types.</p>
<h2>Support crate blocks:</h2>
<ol>
  <li>Chest</li>
  <li>Enchanted Blocks</li>
  <li>Shulker(bugs)</li>
  <li>Barrel</li>
</ol>
